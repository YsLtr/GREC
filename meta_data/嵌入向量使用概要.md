# Steam游戏推荐系统

基于Steam 2025数据集的嵌入向量构建的游戏推荐系统。

## 功能特性

### 1. 基于内容的推荐
- 分析游戏描述文本的语义相似性
- 推荐与用户已有游戏内容相似的新游戏
- 支持排除已拥有游戏

### 2. 基于用户画像的推荐  
- 聚合用户所有游戏的嵌入向量生成用户画像
- 基于用户偏好模式进行个性化推荐
- 使用游戏推荐数作为权重进行加权平均

### 3. 混合推荐算法
- 结合内容相似性和协同过滤信号
- 可调权重参数平衡不同推荐策略
- 基于评测情感分析增强推荐质量

### 4. 多维度相似性计算
- 使用余弦相似度计算向量相似性
- 支持1024维嵌入向量的高效处理
- 考虑游戏质量和受欢迎程度

## 推荐策略对比

| 推荐方式 | 优点 | 适用场景 |
|---------|------|----------|
| 内容相似 | 直观易懂，无需用户历史 | 新用户冷启动 |
| 用户画像 | 个性化程度高 | 有游戏历史的用户 |
| 混合推荐 | 平衡准确性和多样性 | 大多数用户 |

## 使用示例

```python
# 初始化推荐系统
recommender = SteamRecommendationSystem(
    applications_embeddings_path="applications_embeddings.npy",
    reviews_embeddings_path="reviews_embeddings.npy",
    applications_map_path="applications_embedding_map.csv", 
    reviews_map_path="reviews_embedding_map.csv"
)

# 用户已拥有的游戏
user_games = [10, 730, 570, 440]  # CS, TF2, Dota2等

# 基于用户画像推荐
recommendations = recommender.recommend_by_user_profile(user_games, top_k=10)

# 基于特定游戏推荐相似游戏
similar_games = recommender.find_similar_games_by_content(730, top_k=5)  # 找TF2相似游戏
```

## 系统优势

1. **语义理解**: 不仅基于标签，更深入理解游戏内容语义
2. **多语言支持**: 嵌入向量支持跨语言的语义理解
3. **可扩展性**: 易于添加新的推荐算法和特征
4. **实时性**: 支持快速的相似度计算和推荐生成
5. **个性化**: 多层次的用户偏好建模

## 进一步优化建议

1. **时间衰减**: 考虑用户游戏历史的时间权重
2. **负反馈**: 整合用户不喜欢的游戏信息
3. **多模态融合**: 结合图像、音频等其他模态信息
4. **深度学习**: 使用神经网络进一步优化推荐效果
5. **A/B测试**: 对比不同推荐策略的效果
